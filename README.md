# 💰 Simple Interest Calculator

This project is a simple Bash-based calculator that computes **Simple Interest** based on user-provided inputs. It is designed as a beginner-friendly project to demonstrate basic shell scripting, user input handling, and arithmetic operations in Linux/Unix environments.

---

## 📌 Project Description
The Simple Interest Calculator takes three inputs from the user:
- Principal amount (P)
- Rate of interest (R)
- Time period (T)

It then calculates the simple interest using a standard mathematical formula and displays the result in the terminal.

---

## 🧮 Formula Used
Simple Interest (SI) = (Principal × Rate × Time) / 100

---

## ⚙️ Features
- Interactive command-line input
- Fast and lightweight execution
- Beginner-friendly Bash script
- No external dependencies required
- Works on Linux, macOS, and Git Bash (Windows)

---

## 📥 Inputs
- Principal Amount (numeric value)
- Rate of Interest (percentage)
- Time Period (in years)

---

## 📤 Output
- Calculated Simple Interest displayed on the terminal
